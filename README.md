# fast-lorem-ipsum
Generate Lorem Ipsum text quickly

**To use in a project**

$npm install fast-lorem-ipsum
```javascript
var fastLoremIpsum = require('fast-lorem-ipsum').fastLoremIpsum;

fastLoremIpsum("10c");
fastLoremIpsum("100w");

fastLoremIpsum(10,'c');
fastLoremIpsum(100, 'w');

```

**To use from command line**

$npm install fast-lorem-ipsum -g
$loremipsum -c 5
$loremipsum -w 5
